## 🥷 ToDoToDo 설치 및 실행 방법 (macOS)

`ToDoToDo`는 Apple 정식 등록 앱이 아니기 때문에, 다운로드 후 처음 실행할 때 macOS 보안 경고가 뜰 수 있습니다. 당황하지 마시고 아래 순서대로 실행해 주세요!

### 1. 다운로드 및 압축 해제
1. 상단의 `ToDoToDo.zip` 파일을 다운로드합니다.
2. 다운로드된 ZIP 파일의 압축을 풀어 `ToDoToDo.app` 파일을 꺼냅니다.
3. 앱을 **[응용 프로그램(Applications)]** 폴더로 이동시켜 줍니다.

### 2. 첫 실행 시 보안 우회 방법 (중요!)
> ⚠️ **그냥 더블 클릭하면 "악성 소프트웨어 확인 불가"라며 실행이 차단됩니다.** 반드시 아래 방법으로 최초 1회 승인을 해주셔야 합니다.

1. `ToDoToDo.app` 파일 위에서 **마우스 우클릭(또는 트랙패드 두 손가락 클릭)**을 합니다.
2. 메뉴에서 맨 위에 있는 **[열기(Open)]**를 클릭합니다.
3. 경고 창이 다시 뜨지만, 이번에는 없던 **[열기]** 버튼이 새로 나타납니다.
4. **[열기]**를 누르면 이후부터는 더블 클릭만 해도 정상 실행됩니다!

---

### 💡 그래도 안 켜지나요? (터미널 명령어로 강제 해결)
만약 위 방법으로도 차단막이 안 열린다면, 맥북이 이 앱을 격리(Quarantine)해 두었기 때문입니다.

1. **터미널(Terminal)** 앱을 켭니다.
